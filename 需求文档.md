# 厦门旅游推荐系统 - 需求文档

---

## 1. 项目概述

### 1.1 项目名称
厦门旅游推荐系统

### 1.2 项目目标
为游客提供厦门旅游景点查询、天气信息、路线推荐等一站式服务，帮助用户更好地规划厦门之旅。

### 1.3 项目定位
- **类型**：单页应用（SPA）旅游推荐平台
- **用户群体**：计划前往厦门旅游的游客
- **核心价值**：提供景点信息查询、智能路线规划、实时天气查询等功能

## 5. 接口需求

### 5.1 统一响应格式

#### 成功响应
```json
{
    "code": 200,
    "message": "success",
    "data": {},
    "request_id": "uuid"
}
```

#### 错误响应
```json
{
    "code": 400,
    "message": "error message",
    "data": null,
    "request_id": "uuid"
}
```

### 5.2 景点接口

| 方法 | 路径 | 功能 | 参数 |
|------|------|------|------|
| GET | `/api/attractions` | 获取景点列表 | keyword, tag_id[], area, sort_by, sort_order, page, per_page |
| GET | `/api/attractions/:id` | 获取景点详情 | id |
| GET | `/api/attractions/areas` | 获取所有区域 | - |

### 5.3 天气接口

| 方法 | 路径 | 功能 | 参数 |
|------|------|------|------|
| GET | `/api/weather/xiamen` | 获取厦门天气 | city |
| POST | `/api/weather/cache/clear` | 清除天气缓存 | city（可选） |

### 5.4 评论接口

| 方法 | 路径 | 功能 | 参数 |
|------|------|------|------|
| GET | `/api/comments` | 获取景点评论 | attraction_id, page, per_page |
| POST | `/api/comments` | 提交评论 | attraction_id, username, content, rating |

### 5.5 收藏接口

| 方法 | 路径 | 功能 | 参数 |
|------|------|------|------|
| GET | `/api/favorites` | 获取用户收藏 | user_id |
| POST | `/api/favorites` | 添加收藏 | attraction_id, user_id |
| DELETE | `/api/favorites/:id` | 删除收藏 | id |

### 5.6 路线接口

| 方法 | 路径 | 功能 | 参数 |
|------|------|------|------|
| GET | `/api/routes/recommend` | 获取推荐路线 | theme[], duration, weather |
| POST | `/api/routes/custom` | 创建自定义路线 | name, description, theme, attraction_ids[] |

### 5.7 标签接口

| 方法 | 路径 | 功能 | 参数 |
|------|------|------|------|
| GET | `/api/tags` | 获取所有标签 | - |

### 5.8 健康检查接口

| 方法 | 路径 | 功能 | 参数 |
|------|------|------|------|
| GET | `/api/health` | 健康检查 | - |
| GET | `/api/logs` | 获取请求日志 | - |
